# Password Strength Checker

A simple Python project to evaluate the strength of passwords and generate a report.

## Features
- Check password strength based on various criteria
- Generate a report of password strength
- Easy-to-use command-line interface

## Files
- `main.py`: Entry point for running the password strength checker
- `strength_checker.py`: Contains logic for evaluating password strength
- `report.py`: Handles report generation for password strength results

## Usage
1. Clone the repository:
   ```sh
   git clone <repo-url>
   cd password_strength_checker
   ```
2. Run the main script:
   ```sh
   python main.py
   ```
3. Follow the prompts to check password strength and view the report.

## Requirements
- Python 3.7 or higher
